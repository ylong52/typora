#### 未升级版，不需要收费的

win环境使用，经测试可用。不要升级
参考安装路径：
C:\Program Files\Typora\resources

1, 安装exe文件
2. 将C:\Program Files\Typora\resources原有的app.asar改为app.asar.bak
3，将app.asar复制到C:\Program Files\Typora\resources,改名为app.asar
4，启动typora
5, 选【注册】，随意输入邮箱号，任选一个注册号输入
